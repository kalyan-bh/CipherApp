Explaining Monoalphabetic Cipher

Monoalphabetic Cipher is a substitution cipher where each letter in the plaintext is mapped to another letter in a one-to-one fashion. Let's see an example of how it works:

Suppose we want to encrypt the message "HELLO" using a Monoalphabetic Cipher with the following substitution mapping:

Plain Text: HELLO

Substitution Mapping:

A - H

B - E

C - L

D - O

E - R

...

In this example, each letter in "HELLO" is substituted according to the mapping provided, resulting in the cipher text.

Decryption in Monoalphabetic Cipher involves knowing the substitution mapping used for encryption. By reversing the substitution, we can decrypt the message.

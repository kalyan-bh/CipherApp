Explaining Caesar Cipher

Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet. Let's see an example of how it works:

Suppose we want to encrypt the message "HELLO" using a Caesar Cipher with a shift of 3. Here's how it would look:

Plain Text: HELLO

Shift: 3

Cipher Text: KHOOR

In this example, each letter in "HELLO" is shifted 3 positions down the alphabet, so "H" becomes "K", "E" becomes "H", and so on.

The decryption process is simply the reverse of encryption. If we have the cipher text "KHOOR" and we know it was encrypted using a Caesar Cipher with a shift of 3, we simply shift each letter 3 positions up the alphabet to get back the original message "HELLO".
